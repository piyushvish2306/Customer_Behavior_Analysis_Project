# 📊 Customer Behavior Analysis Project

---

## 🔹 Project Overview  
This project focuses on analyzing customer behavior using **Python, SQL, and Power BI** to uncover meaningful insights from shopping data.  
The goal is to help businesses understand customer patterns, improve decision-making, and optimize sales strategies.

---

## 🎯 Business Problem  
Businesses often struggle to understand:
- Customer purchasing patterns  
- High-value vs low-value customers  
- Product performance across categories  
- Customer segmentation  

👉 This project solves these challenges using data-driven analysis.

---

## 🛠️ Tech Stack  
- **Python** → Data Cleaning & EDA (Pandas, NumPy, Matplotlib)  
- **SQL (MySQL)** → Data Analysis & Querying  
- **Power BI** → Interactive Dashboard  
- **Jupyter Notebook** → Development Environment  

---

## 🔄 Project Workflow  

1. Import data into Python  
2. Perform Data Cleaning & EDA  
3. Load data into SQL database  
4. Analyze data using SQL queries  
5. Build interactive dashboard in Power BI  
6. Summarize insights  
7. Upload project to GitHub
